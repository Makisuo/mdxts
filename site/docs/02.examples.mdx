# Examples

Examples can be written several ways. Choose the one that best fits your needs. You can force a specific method for writing examples by setting the `examples` option in your `mdxts.config` file.

## Local

Local examples can be exported alongside your component using the [[example]] helper:

```tsx filename=Button.tsx
import { example } from 'mdxts'

/** Used for taking actions and navigation. */
export function Button({
  label,
  onAction,
}: {
  /** Provides a label for the button. */
  label: string

  /** Called when an action for this button is requested. */
  onAction: () => void
}) {
  return <button onClick={onAction}>{label}</button>
}

/** Used for taking actions and navigation. */
example(() => <Button label="Say hello" onAction={() => alert('Hello!')} />)
```

## External

Export multiple examples using the `examples` extension:

```
Button.tsx
Button.examples.tsx
```

```tsx filename=Button.examples.tsx
import { Button } from './Button'

export function BasicUsage() {
  return <Button label="Say hello" onAction={() => alert('Hello!')} />
}

export function DifferentSizes() {
  return ['small', 'medium', 'large'].map((size) => (
    <Button label="Say hello" size={size} />
  ))
}
```

Or group examples into a directory:

```
Button/
├── index.tsx
├── exampels/
│ ├── example.01.tsx
│ ├── example.02.tsx
│ ├── example.03.tsx
```

## Comments

JSDoc comments can be used to document examples. All examples will be compiled and available in the returned object.

```tsx
/**
 * Say hello.
 *
 * @example
 * <Hello name="Penny" />
 */
function Hello({ name }: { name: string }) {
  return <div>Hello, {name}</div>
}
```

Gather all `@example` tags from a file and render them in MDX:

```tsx Button/index.tsx
/**
 * Provides a button for taking actions and navigating.
 *
 * @example
 * <Button label="Say hello" onPress={() => alert('Hello!')} />
 */
export function Button({
  label,
  onPress,
}: {
  label: string
  onPress: () => void
}) {
  return <button onClick={onPress}>{label}</button>
}
```

```mdx filename=Button/README.mdx
<Example identifier="Button" />
```
